# Project Instructions Template

## Purpose

Define stable rules that should guide every AI-assisted task in the repository.

## Template

```text
Project: [NAME]
Stack: [STACK]
Architecture rules: [RULES]
Coding standards: [STANDARDS]
Testing expectations: [TESTS]
Security constraints: [SECURITY]
Accessibility constraints: [A11Y]
Do not: [ANTI_PATTERNS]
Escalate when: [ESCALATION]
```

## Principle

Context should be short enough to stay usable, but complete enough to prevent architectural drift.
