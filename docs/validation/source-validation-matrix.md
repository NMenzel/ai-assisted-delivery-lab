# Source Validation Matrix

## Validation Categories

- Framework APIs: verify against installed framework docs or official docs.
- Security guidance: verify with security owner or approved internal policy.
- Accessibility guidance: verify against WCAG sources and human expert review.
- Legal and compliance claims: verify with qualified counsel or compliance owner.
- Tool behavior: verify in the local project or vendor documentation.

## Language Rule

Use source-informed, expert-review-ready, prepared for human expert approval, or human-reviewed once reviewed. Do not claim expert-approved, certified, compliance-guaranteed, or legally validated without documented approval.
