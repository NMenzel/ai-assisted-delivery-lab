# AI-Assisted Accessibility Audit Workflow

AI supports accessibility audit workflows. It does not replace human accessibility expertise, manual testing, or final compliance judgment.

## Where AI Can Help

- Triage likely WCAG issues.
- Draft test plans.
- Summarize repeated findings.
- Suggest remediation options.
- Draft client-facing explanations.

## Where Human Review Is Mandatory

- Manual keyboard testing.
- Screen reader validation.
- WCAG interpretation.
- Severity assignment.
- Compliance or conformance statements.
- Final client deliverables.

## Risks

- False positives.
- False negatives.
- Incomplete assistive technology coverage.
- Missing context from screenshots or code snippets.
