# AI-Generated Code Review Checklist

Review AI-generated code for:

- Architecture fit
- SOLID
- DRY
- KISS
- Separation of concerns
- Type safety
- Error handling
- Security
- Accessibility
- Performance
- Test coverage
- Dependency risk
- Maintainability
- Observability
- Documentation
- Hallucinated API detection
- Overengineering detection
- Hidden behavior changes

## Review Rule

AI-generated code must meet the same bar as human-written code. If provenance is unclear, review it as untrusted until verified.
