# AI Workflow Evaluation Framework

## Metrics

- code quality
- review quality
- bug detection
- security issue detection
- accessibility issue detection
- documentation quality
- token efficiency
- developer time saved
- hallucination rate
- human correction rate
- rework rate

## Monthly Review Cycle

1. Select benchmark tasks.
2. Run approved prompts.
3. Score outputs.
4. Record hallucinations and corrections.
5. Retire or revise weak prompts.
6. Update training examples and policy notes.
