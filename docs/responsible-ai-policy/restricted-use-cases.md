# Restricted Use Cases

AI assistance is restricted when mistakes can create material security, legal, accessibility, financial, operational, or customer harm.

## Restricted Categories

- Security controls
- Authentication and authorization
- Secrets handling
- Production incident response
- Legal and compliance interpretation
- Accessibility conformance claims
- Medical or safety-related advice
- Destructive system actions
- Customer-impacting automated decisions

## Required Controls

- Explicit human owner.
- Narrow prompt scope.
- Approved data handling.
- Review checklist.
- Test or evidence trail.
- Clear final sign-off.
