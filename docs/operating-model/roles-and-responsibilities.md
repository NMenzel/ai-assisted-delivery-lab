# Roles And Responsibilities

## Engineer

The engineer owns correctness, maintainability, test evidence, and final implementation decisions. AI-generated code must be reviewed as if it came from a junior contributor with no project context unless proven otherwise.

## Reviewer

The reviewer checks architecture fit, SOLID, DRY, KISS, separation of concerns, error handling, observability, maintainability, and hidden behavior changes.

## Security Reviewer

The security reviewer checks data handling, secrets exposure, authentication and authorization changes, dependency risk, prompt injection exposure, and excessive tool permissions.

## Accessibility Reviewer

The accessibility reviewer checks semantics, keyboard behavior, focus management, readable labels, contrast, manual test evidence, and false positive or false negative risk.

## Product Owner

The product owner confirms that scope, user impact, and release timing are acceptable.
