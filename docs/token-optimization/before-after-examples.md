# Before And After Examples

## Before

```text
Please review this code and tell me everything you think could be better, including style, architecture, tests, and any other improvements.
```

## After

```text
Review for correctness, security, accessibility regressions, and missing tests. Findings only. Include file references.
```

## Tradeoff

The shorter prompt reduces noise and token cost, but it should not be used when broad architecture exploration is required.
